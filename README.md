# Slides Presentation with Svelte + Reveal.js

## Creating a presentation

Add your slides to `src/lib/presentation.svelve`

## Running

```bash
# or start the server
pnpm dev

# or start the server and open the app in a new browser tab
pnpm dev --open
```
